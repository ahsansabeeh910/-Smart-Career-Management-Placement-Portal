# 📌 Overview

The **Campus Recruitment System** is a Python-based placement management portal developed using **Streamlit** for simplifying and automating the campus recruitment process.

The platform acts as a centralized system connecting:

- Students
- Companies
- Administrators

The system streamlines:

✅ Job Posting  
✅ Job Applications  
✅ Recruitment Tracking  
✅ Student Management  
✅ Company Management  
✅ Placement Workflow Automation  

The project was developed as part of the **Software Engineering Lab** at **Jaypee Institute of Information Technology**. :contentReference[oaicite:1]{index=1}

# 🧠 System Architecture

The system uses role-based authentication and dashboard routing:

```text
Student  → Apply for jobs
Company  → Post jobs & manage applications
Admin    → Manage entire platform
```
